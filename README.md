# Binary Gap

### Task
Find the maximal length of a sequence of zeroes surrounded by ones in the binary representation of a positive number.

### Examples
#### Example 1
_n=4_  
_bin(n)=100_  
_result=0_

#### Example 2
_n=5_  
_bin(n)=101_   
_result=1_

#### Example 3
_n=2,147,483,619_  
_bin(n)=1111111111111111111111111100011_   
_result=1_
